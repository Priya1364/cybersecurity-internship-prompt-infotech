# Cybersecurity Internship - Prompt Infotech

## Overview

This repository contains all assignments, practical exercises, screenshots, reports, and learning outcomes completed during my 15-Day Cybersecurity Internship at Prompt Infotech.

The internship focuses on:

* Network Fundamentals
* Packet Analysis using Wireshark
* Protocol Analysis
* DNS and ICMP Investigation
* Network Configuration
* Vulnerability Assessment
* Information Gathering Techniques
* Practical Cybersecurity Concepts

* **Internship Domain:** Cybersecurity
* **Organization:** Prompt Infotech

## Learning Objectives

* Understand network communication.
* Analyze packets using Wireshark.
* Learn TCP/IP protocols.
* Study DNS and ICMP traffic.
* Perform basic information gathering.
* Understand NAT and Bridged networking.
* Capture and analyze HTTP traffic.
* Gain practical cybersecurity experience.
